VERSÃO ALTA QUALIDADE — VÍDEOS COM PRELOAD NONE

Esta versão mantém os vídeos originais em melhor qualidade visual,
mas altera os players para:

preload="none"

Isso significa:
- a página não baixa os vídeos automaticamente ao carregar;
- os vídeos só carregam quando o usuário toca no play;
- a qualidade visual fica muito melhor que a versão super compactada.

Observação:
O pacote fica maior no GitHub, mas a experiência inicial da página fica protegida pelo preload="none".

Preservado:
- GTM
- v18
- protocolo MRG-XXXXXX
- WhatsApp
- captura de UTMs
- copyright
- seção de prova social em vídeo
